# Penguin Magic 魔改版 v0.3.0 更新说明

**版本**：v0.3.0  
**基于**：魔改 v0.2.0  
**说明**：本版本以「素材库与创意文本库」为核心，画布 Tab 优先、浮动面板、点击/拖拽放置、按文件夹与画布分类，并优化画布右键与保存/删除交互。

---

## 本次更新概要

### 素材库与创意文本库

- **画布 Tab 优先**：应用启动/导航后默认选中画布 Tab，减少一步点击。
- **浮动面板**：
  - 素材库、创意文本库由固定侧边栏改为画布内的**浮动面板**，可拖拽、收起为小图标、锁定、边缘拉伸调整大小。
  - 画布内创意库同样为浮动面板，行为一致。
- **放置方式**：
  - **点击放置**：在素材库中点击素材，在画布上点击目标位置即可添加图片节点。
  - **拖拽放置**：从素材库拖拽素材到画布空白处，释放后生成图片节点。
- **分类**：素材与创意支持按**文件夹**与**画布**维度分类展示与管理。
- **右键与保存/删除**：画布内右键菜单与「保存/删除」弹窗逻辑调整，避免误触；右键不再默认弹出保存/删除。

### 画布与设置

- **画布**：CanvasNode、PebblingCanvas 与 ImageGenPanel 等组件配合素材库放置与生成流程做了调整与精简。
- **设置**：SettingsModal 与 API/主题等配置的展示与交互优化。
- **后端**：文件路由与缩略图工具增强，支持素材库列表、分类与预览需求。

## 使用说明

- **打开素材库/创意库**：进入画布后，在左下或右侧找到浮动图标，点击展开；可拖拽面板位置、点击锁定、拖动边缘调整大小。
- **添加素材到画布**：在素材库中点击一项再在画布上点击，或直接拖拽素材到画布空白处。
- **分类查看**：在素材库/创意库中使用「按文件夹」「按画布」等分类筛选。
