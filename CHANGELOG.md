# 更新日志 Changelog

本文件记录本项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.3.8] - 2026-02-11

### 改进
- 图片生成/编辑 API 增加图片压缩功能，解决多图输入时可能触发的 128MB Payload Too Large 报错问题
- 优化 Gemini 与多模态服务的调用逻辑，提升稳定性
- Pebbling Canvas 画布节点样式调整与交互改进，颜色系统优化

### 新增
- Pebbling Canvas 新增 ComfyUI 插槽输入、自定义选择器、预览缩略图等多个核心组件
- 后端支持更复杂的画布 API 调用流程

## [0.3.2] - 2026-02-06


### 改进
- 图片节点生成配置弹窗：多图输入时按顺序展示预览，每张预览图右上角显示顺序标签（1、2、3…），便于多图描述时对应顺序，交互参考可灵 O1 节点（仅顺序标签，无名称与插入）

## [0.3.0] - 2026-02-06

### 新增
- 画布 Tab 优先：进入应用后默认选中画布 Tab
- 素材库与创意文本库改为画布内浮动面板，支持拖拽、收起、锁定、边缘调整大小
- 素材库点击/拖拽放置：点击素材后在画布点击放置，或拖拽素材到画布添加图片节点
- 素材与创意按文件夹、画布分类展示与管理

### 改进
- 画布内右键与保存/删除弹窗逻辑调整，避免误触
- 画布节点、ImageGenPanel、CreativeLibrary、SettingsModal 等组件与浮动面板及放置流程优化
- 后端文件路由与缩略图接口扩展，支持素材库与分类

### 说明
- 详见 ANNOUNCEMENT_v0.3.0.md、RELEASE_NOTES_v0.3.0.md

- 画布拖拽事件与渲染稳定性优化
- ComfyUI 相关组件与 API 调整

### 移除
- 移除 README 顶部图片引用（不再随仓库备份）
- 移除旧版按版本号命名的 CHANGELOG 文件（CHANGELOG_1.x.x.txt），统一使用本文件 CHANGELOG.md

## 版本号说明

- **主版本号 (MAJOR)**：不兼容的 API 修改或重大架构变更
- **次版本号 (MINOR)**：向下兼容的功能性新增
- **修订号 (PATCH)**：向下兼容的问题修正

## 变更类型

- **新增**：新功能
- **改进**：对现有功能的改进
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **问题修复**：Bug 修复
- **安全**：安全相关的修复
